# Tommy Na Hub

The official personal hub for Tommy Na. This site serves as a landing page for my social media, professional links, and shop.

## About
Cardiac Sonography student, content creator, and fantasy sports enthusiast. This project is a custom-built, lightweight personal hub designed to centralize my digital presence.

## Tech Stack
- **Frontend:** HTML5, CSS3
- **Deployment:** Cloudflare Pages
- **Version Control:** GitHub

## How to Update
To add new links or change existing ones:
1. Edit `index.html` in your local text editor.
2. Commit your changes and push to the `main` branch.
3. Cloudflare will automatically detect the push and redeploy the site.
